NEURAL WAVETABLE GENERATOR

What is this?

This wavetable generator is based on a simple autoencoder I developed some time ago, which has been trained on a dataset of 15000 waveforms (each 600 samples long). The latent space of the autoencoder is 8-dimensional, and the sliders above control the values of these latent parameters. The neural network then decodes these latent parameters into a new waveform, which is displayed in the chart above. You can adjust the sliders to explore the latent space and create new sounds. The "Smooth" button applies a simple lowpass filter to the waveform, while the "Window" button applies a Hann window to the waveform.

You can download single-cycle waveform in WAV format (to be used in the many synths and samplers supporting it), also adjusting length, bit depth, and sample rate.

An earlier version of the network was used to create the wavetable generators which the Max for Live devices of Neuraloot package are based on. You can also find a neuralwave based oscillator Max/MSP external in my open source collection sonus.
